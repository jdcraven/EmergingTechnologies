Project Details

In a previous class, Advanced Open Source web development, we had briefly touched on using the MEAN development method using Mongo DB as the database and using Express JS and Node JS for the server using the Angular JS to handle the front end. We were only able to briefly touch on using Angular JS and I would like to further my knowledge using all these tools especially Angular JS for at my work we have been starting to implement Angular into our projects, however in a Microsoft .Net environment at work with an sql server DB. I hope to learn more about making the website secure and publicly accessible and security/managing Mongo DB also learning more about Angular JS. I would like to build an application that will allow user to login to the site using their Facebook accounts and be able to track their exercise. Users will be able to track their progress of several excercises for example running, pushups, sit ups, or weights lifting exercises and the reps and weights that they did.


Application Details

The application will be able to:
  • Track different exercise regiments and track weight loss/gains
  • Search by exercise, date etc.
  • Share progress with Friends on Facebook
